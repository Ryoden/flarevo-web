構成


監視カメラ
現場に設置するカメラはネットワークカメラを利用します。画質が良く数多くのカメラがつけられる特徴があります。
録画サーバー
パソコンやサーバー機器を利用して監視カメラの映像を録画します。録画サーバーはカメラ台数によって複数台準備いたします。
閲覧パソコン
閲覧はネットワークにつながれたパソコンを利用して行います。ネットワークに接続していれば、どのパソコンからも閲覧可能です。

Nx Server ― 録画サーバー

パワフルな録画ソフトウェア
大量のカメラ映像を同時録画できる軽量設計
自動フェールオーバー機能標準搭載
WindowsおよびLinuxに対応

主な特徴
カメラとサーバーの自動検索
120を超えるメーカーのカメラとデバイスに対応 (e.g. ONVIF IP Cameras, I/O devices, storage)
1台から100台以上の幅広いインテグレーション
詳細な権限管理と証跡管理
開発ツール（API/SDK）無償提供
サードパーティシステムとの連動

Nx Desktop ― 閲覧パソコン

快適さを追求した閲覧ソフトウェア
メディアプレイヤーを元に作成された閲覧ソフト
複数映像を途切れることなく同時再生
直感的な操作とわかりやすいインターフェース

主な特徴
ライブ映像と録画映像の閲覧（同時閲覧も可）
自由なレイアウトと自動配置
拡大 / 縮小 / 回転 表示
モーション検索
双方向音声
WEBページの表示
WindowsおよびMacに対応
